iOS Indexing Checker for Windows - One-click package

English

1. Extract the ZIP file.
2. Double-click Start-iOS-Indexing-Checker.cmd.
3. Choose English or Chinese.
4. Connect the iPhone by USB.
5. Unlock the iPhone and tap Trust This Computer if asked.
6. Open Settings on the iPhone.
7. Return to the tool window and press Enter.

This package includes the offline core program. You do not need to install Python.

Requirements:

- Windows can recognize the iPhone.
- Apple Devices or iTunes is installed if this PC has never connected to an iPhone before.

Privacy:

The tool reads live iPhone logs locally and does not upload logs.

If something goes wrong, ios-indexing-checker.log may be created in this folder for troubleshooting.

中文

1. 解压 ZIP 包。
2. 双击 Start-iOS-Indexing-Checker.cmd。
3. 选择 English 或中文。
4. 用 USB 线连接 iPhone。
5. 解锁 iPhone，如果弹出提示，点 信任此电脑。
6. 在 iPhone 上打开 设置。
7. 回到工具窗口，按 Enter。

这个包已经包含离线核心程序，不需要安装 Python。

需要：

- Windows 能识别 iPhone。
- 如果这台电脑以前从没连过 iPhone，请先安装 Apple Devices 或 iTunes。

隐私：

工具只在本机读取 iPhone 实时日志，不会上传日志。

如果出现问题，同目录下可能会生成 ios-indexing-checker.log，用于排查。
