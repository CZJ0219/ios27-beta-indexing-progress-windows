# 隐私说明

本工具的目标很窄：读取本机连接的 iPhone 实时日志，并从中查找索引进度。

## 会读取什么

工具只关心和索引进度有关的日志行，并把找到的百分比显示出来。

## 会保存什么

启动器会在工具目录保存一个本地日志：

```text
ios-indexing-checker.log
```

日志用于排查问题，可能包含设备名、iOS 版本、进度日志等信息。

## 不会做什么

- 不上传日志。
- 不连接项目作者的服务器。
- 不收集遥测。
- 不修改 iPhone 数据。
- 不读取聊天记录、照片、通讯录或备份内容。

## 分享日志前

如果你要在 GitHub Issue 或公开场合贴日志，请先遮盖：

- 设备名。
- UDID。
- Apple ID。
- 邮箱、手机号。
- 任何你不想公开的路径或账号名。
